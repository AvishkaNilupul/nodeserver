3 · Export

Ready-to-import JSON (matches the account-pool importer: Login, ClientSecret, Id, UniqueId). Only integrity-verified accounts are included by default.

Heads-up on re-auth: the account-pool importer fills a ClientSecret onto a row only when that row doesn't already have one — so importing a fresh token will not overwrite an existing (bad) token on a pool account. For brand-new accounts it just works. To replace a bad token on an existing pool row, delete that row first (or ask for a "force re-auth" import toggle).
